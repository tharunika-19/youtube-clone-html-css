# 🎬 YouTube Homepage Clone

A simple **YouTube Homepage Clone** built using **HTML and CSS**.  
This project recreates the layout of the YouTube homepage including the **navbar, sidebar, categories section, and video grid layout**.

---

## 🚀 Live Demo

You can view the project here:

👉 https://tharunika-19.github.io/youtube-clone-html-css


## 📌 Features

- Responsive **YouTube-like layout**
- Fixed **navigation bar**
- Expandable **sidebar menu**
- **Video grid layout** similar to YouTube
- Hover effects on video cards
- Video duration badge
- Category filter buttons

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **Flexbox**
- **CSS Grid**

## 🧠 Algorithm / Development Steps

1. Create the **basic HTML structure**.
2. Build the **navigation bar** with logo, search bar, and icons.
3. Create the **sidebar menu** for navigation options.
4. Design the **category filter section**.
5. Implement the **video grid layout** using CSS Grid.
6. Add **video thumbnails and channel icons**.
7. Include **hover effects and video duration badges**.
8. Style the layout using **CSS Flexbox and Grid**.
